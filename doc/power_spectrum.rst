PowerSpectrum
======================

.. py:class:: PowerSpectrum

   .. py:attribute:: n

   Number of k bins; length of k, P0, P2, P4 array.
   Equivalent to `len(ps)`
		     
   .. py:attribute:: k

   Array of wave numbers [h/Mpc]

   .. py:attribute:: nmodes

   Array of number of independent modes in bins

   .. py:attribute:: P0

   Array of power spectrum monopoles

   .. py:attribute:: P2

   Array of power spectrum quadrupoles

   .. py:attribute:: P4

   Array of power spectrum hexadecapoles
